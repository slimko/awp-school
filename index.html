Необходимые теги в Open Graph

  • og:title - тег, отвечающий за название материала;
  • og:description – тег, содержащий описание материала (причём в отличие от традиционного мета-тега description стандарт Open Graph позволяет увеличить размер описания со 160 до 295 знаков);
  • g:image – ссылка на картинку, которая должна сопровождать материал;
  • og:type – тег, указывающий на тип добавляемого материала, например, «article» – статья, «movie» – кино и т.д.;
  • og:url – ссылка на саму веб-страницу, которая добавляется в социальную сеть

2. Анимации

Пример Анимации над div

Иван Марков

изучал по статье: https://html5book.ru/css3-animation

и по статье: http://frontender.info/ochen-prostoe-rukovodstvo-po-css-animatsiyam/

Анимация представляет собой набор ключевых кадров, который содержится в css

Основные свойства:

  • @keyframes - Ключевое слово указывает на саму анимацию.
  • animation-name - Название анимации
  • animation-duration - Продолжительность анимации
  • animation-timing - function Временная функция
  • animation-delay - Анимация с задержкой
  • animation-iteration-count - Повтор анимации
  • animation-direction - Направление анимации
  • animation - Краткая запись анимации (Все параметры воспроизведения анимации можно объединить в одном свойстве)
  • animation-play-state - Проигрывание анимации
  • animation-fill-mode - Состояние элемента до и после воспроизведения анимации

3. Поддержка Retina

Разницу в изображении на Retina дисплеях мне удалось увидеть только у товарища на mac'е, исходя из того, что я прочитал существует ряд способов для внедрения поддержки retina для изображений на сайте

  • Определить плотность пикселей экрана в медиазапросе конструкцией device-pixel-ratio, при которой подгружаем картинку в два раза большую
  • изначально загружать графику в большом размере и принудительно (задавая параметры width и height) уменьшать вдвое
  • использовать для распознавания плотности пикселей экрана Javascript и заменять картинки
  • использовать масштабируемую векторную графику - SVG
  • Применительно для элементов использовать - Иконочные шрифты
